## Trilinos Versions

Even point version numbers, e.g. 10.0.0, 10.0.1, 10.2.0, 10.2.3, indicate release versions of Trilinos. Odd point version numbers, e.g. 10.1, 10.3, 10.5, indicate development (non-release) versions of Trilinos.
